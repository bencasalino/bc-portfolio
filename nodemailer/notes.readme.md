# nodemailer Gmail Example

## Setup

```sh
cp .env.sample .env
```

* Update the .env file with your gmail account and the from/to account
* Create an app password and add it to your .env
  * https://support.google.com/mail/answer/185833

## Run

```sh
npm install
npm start
```
